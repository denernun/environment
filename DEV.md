**NVM**
---

    https://github.com/coreybutler/nvm-windows

    Install in D:\NVM
    Install SymLynk NodeJS in D:\NODE

    Root
    
    D:\NVM
    
    Path

    D:\NVM
        
    Comands
    
    nvm root <path> [D:\NVM]
    nvm install <version> arch [32|64]
    nvm list
    nvm use <version>
    nvm uninstall <version>
